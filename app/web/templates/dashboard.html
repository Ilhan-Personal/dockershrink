Next Step: Install & Run the CLI

1. Download the DockerShrink CLI for your platform from the Releases page .

If you're using MacOS, install the CLI via Homebrew instead:
brew install duaraghav8/tap/dockershrink
‍
2. Copy your DockerShrink API Key from above to initialize the CLI

dockershrink init --api-key <paste your api key here>
‍
3. Navigate to the root directory of your nodejs project and run Dockershrink!

dockershrink optimize
# Optionally, provide your openai api key to eable AI features
dockershrink optimize --openai-api-key <your openai api key>
‍
4. Don't forget to share your feedback with us!
    We're actively working to improve DockerShrink and are very excited to hear what you think.
