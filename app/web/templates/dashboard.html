Next Step: Install & Run the CLI

1. Download the DockerShrink commandline application from our Github Repository
2. Use your DockerShrink API Key to initialize the CLI

dockershrink init --api-key <paste your api key here>
‍
3. Navigate to the root directory of your project and invoke the dockershrink cli

dockershrink
dockershrink --openai-api-key <your openai api key>
‍
4. Don't forget to share your feedback with us!
    We're actively working to improve DockerShrink and are very excited to hear what you think.
